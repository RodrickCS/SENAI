introdução do sistema

Este sistema tem como função gerenciar os produtos e outros serviços de uma loja.


Ao lado terá um menu com as várias opções de gerenciamento, como:


  • Gerenciar Produtos
  • Gerenciar Setores
  • Gerenciar vendedores

Na sessão Gerenciar Produtos é possivel adicionar novos produtos de acordo com o setor

Na sessão Gerenciar Setores é possivel adicionar novos Setores

Na sessão Gerenciar Vendedores é possivel adicionar novos vendedores
